O seu time ficou responsável pela construção desse fluxo de ingestão de dados e você pelo módulo de Data Quality deste fluxo. 
Este módulo vai escutar uma fila,
validar os eventos trafegados nela a partir de JSONs Schemas armazenados em um repositório, 
e enviar para uma outra fila (valid-events-queue).

Construa esse módulo em python a partir dos arquivos disponibilizados para você na pasta "exercicio1". Você pode ficar à vontade para propor melhorias, mas fique atento aos pontos a seguir:

Use o JSON Schema disponibilizado na pasta do exercício (schema.json) para validar o evento, sem o auxílio de bibliotecas externas de validação, e garantindo que:

i. Tipo do campo do evento deve bater com o do schema.

ii. Não deve aceitar campos não cadastrados no schema.

Evento de saída deve ser igual ao de entrada.

Desenvolva seu código no arquivo "event_validator.py". É a função "handler" deste arquivo que será estimulada sempre que um novo evento cair na fila e utilize a função "send_event_to_queue" para enviar eventos.

Execute 'python main.py' para simular o fluxo.
